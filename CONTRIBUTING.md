# Contributing

Thanks for your interest in contributing! This project is research-focused and
intended for robustness testing only. Please keep that scope in mind.

## Before you start
- Read the README for setup, training, and evaluation workflows.
- Open an issue to discuss large changes.

## How to contribute
1) Fork the repository and create a feature branch.
2) Make your changes with clear, minimal diffs.
3) Ensure linting/tests (if any) still pass.
4) Submit a pull request with a clear description and rationale.

## Coding guidelines
- Prefer small, readable changes.
- Keep comments focused on non-obvious logic.
- Avoid introducing heavy dependencies without discussion.

## Research safety
This repository is for research and robustness testing only. Do not use it to
cause harm or unsafe behavior.

## Reporting issues
When filing an issue, include:
- Your OS, Python version, and GPU/CPU info
- Exact commands you ran
- Logs and error traces
- Steps to reproduce
